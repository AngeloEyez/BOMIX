# 版本更新記錄

本文件記錄 BOMIX 的所有重要變更。格式基於 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.1.0/)。

## [0.2.0] - 2026-02-13

### 變更
- 框架遷移：Python/Flet → Electron + React + Vite + Tailwind CSS v4
- 新增 TanStack Table 作為表格元件
- 新增 Zustand 狀態管理
- 資料庫改用 better-sqlite3
- Excel 處理改用 xlsx (SheetJS)
- 打包工具改用 electron-builder
- 新增 Jules + Antigravity 雙 Agent 協作規範
- CI/CD 從 Python 改為 Node.js
- 新增 VS Code Tasks 取代自訂建置腳本

## [0.1.0] - 2026-02-12

### 新增
- 初始化專案目錄結構
- 建立基礎開發環境設定
