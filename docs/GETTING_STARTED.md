# BOMIX 快速上手指南

## 安裝

1. 下載最新版本的 `BOMIX Setup.exe` 或 `BOMIX.exe`（便攜版）
2. 安裝版：執行安裝程式，按照提示完成安裝
3. 便攜版：將檔案放置在任意目錄，雙擊執行

## 第一次使用

### 建立新系列
1. 在首頁點選「建立新系列」
2. 選擇儲存位置與檔案名稱
3. 系統會建立一個新的資料庫檔案（`.bomix`）

### 匯入 BOM
1. 在專案中選擇目標 Phase 與版本
2. 點選「匯入」按鈕
3. 選擇 Excel BOM 檔案（支援 `.xls` 與 `.xlsx`）
4. 確認匯入結果

## 備份資料

直接複製 `.bomix` 檔案即可完成備份。
