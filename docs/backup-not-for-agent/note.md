```
/goal 根據 `roadmap.md` 與 `CLAUDE.md` 的技術規範，啟用 `phase-executor` 子代理全自動推進開發。

3. 除非遇到以下兩種情況，否則【禁止中斷】或詢問我：
   - 產品規格或邏輯出現嚴重衝突，需要我進行決策。
   - 子代理重試超過 5 次仍卡在同一個編譯/測試阻礙。
4. 目標終點：直到 `roadmap.md` 中所有 Phase 的項目全部變成 `[x]`，且最終專案可成功編譯。
```
